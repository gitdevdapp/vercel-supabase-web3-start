================================================================================
WHY THIS FIX WORKS
================================================================================

PROBLEM: 
- Flexbox with flex-grow (flex-1) + min-width constraints = unpredictable sizing
- Icon (4px) + margin (2px) + varying text lengths = text wrapping
- "Transaction History" text too long for constrained button width

SOLUTION:
- CSS Grid with single column (grid-cols-1) = one button per row
- Full width (w-full) = predictable sizing, no grow conflicts
- Text truncation (<span class="truncate">) = graceful overflow handling
- Icon non-shrinking (flex-shrink-0) = consistent icon placement

RESULT:
- Each button on its own line
- Consistent spacing (gap-2 = 8px)
- Proper text alignment
- Works on all screen sizes
- No responsive breakpoints needed
